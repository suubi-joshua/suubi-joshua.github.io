About

Building software that
serves real people.

I'm a software developer based in Kampala, Uganda, focused on digital health infrastructure. My work spans the full stack — from designing intuitive frontends with React to building resilient backends with Java and Spring Boot.

I contribute to OpenMRS, the world's largest open-source health platform, and build tools that help clinicians deliver better care — even in low-resource settings with limited connectivity.

Beyond code, I'm passionate about Linux server administration, Docker deployments, and crafting developer workflows that just work.

I believe open source is the most powerful lever for solving problems at scale — and that good software should work for everyone, especially those who need it most.
